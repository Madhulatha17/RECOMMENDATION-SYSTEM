# RECOMMENDATION-SYSTEM

## Project Description
This project implements a Recommendation System using Collaborative Filtering (Matrix Factorization - SVD).

It uses the MovieLens dataset and predicts user ratings for unseen items.

##  Features
- Collaborative Filtering
- Matrix Factorization (SVD)
- RMSE & MAE Evaluation
- Cross Validation
- Top-N Recommendations

## Algorithm Used
- Singular Value Decomposition (SVD)

## Evaluation Metrics
- RMSE (Root Mean Square Error)
- MAE (Mean Absolute Error)
